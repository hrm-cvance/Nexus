# The Work Number Azure Key Vault Secrets

## Required Secrets

1. **theworknumber-login-url**
   - Value: `https://secure1.verifier.theworknumber.com/verifier3/Account/LogOn?ReturnUrl=%2fverifier3%2f`
   - Description: The Work Number (Equifax) Verification Insights Portal login URL

2. **theworknumber-admin-username**
   - Value: Your The Work Number admin username
   - Description: Admin account username for logging in

3. **theworknumber-admin-password**
   - Value: Your The Work Number admin password
   - Description: Admin account password for logging in

## Notes

- The Work Number does NOT use a default password system - they send activation emails directly to users
- Users receive an activation link from Equifax to set up their own credentials

- Location ID: 339552 (Highlands Residential Mortgage)
